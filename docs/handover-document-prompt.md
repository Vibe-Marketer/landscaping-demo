# Handover Document Generation Prompt

You are a professional documentation specialist tasked with creating a comprehensive handover document based on a conversation thread. This handover document will be used to transfer complete knowledge and context to another person who needs to seamlessly continue the work without any gaps in understanding or need for clarification.

**Task:**

Create a detailed, structured handover document that captures every aspect of the conversation thread, including all decisions made, context provided, work completed, outstanding items, and next steps. The document must be comprehensive enough that a new person can immediately understand the full scope and continue the work to completion without asking any questions.

**Objective:**

Ensure complete knowledge transfer and project continuity by providing a handover document that eliminates any information gaps and enables seamless transition of responsibilities to another team member.

**Knowledge:**

- Review the entire conversation thread systematically from beginning to end
- A complete handover document should include: project background, key decisions made, work completed, current status, outstanding deliverables, next steps, important context, stakeholder information, deadlines, dependencies, and any critical details discussed
- The receiving person should have zero need to reference the original conversation thread after reading this document
- Include specific details, not generalizations, to ensure accuracy and completeness
- Structure the information logically and chronologically where relevant
- Highlight any urgent items, blockers, or time-sensitive elements
- Your life depends on you ensuring no critical information is omitted from this handover document, as incomplete knowledge transfer could result in project failure or significant delays

**Output Requirements:**
- Save prompt to: `docs/handover-document-prompt.md`
- Save handover document to: `example-handover.md` (root directory)
