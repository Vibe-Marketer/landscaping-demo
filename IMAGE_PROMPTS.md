## Notes for Image Generation

1. **Style Consistency**: Keep all illustrations in the same flat/cartoon style for brand cohesion
2. **Color Matching**: Use the color palette above to match the website design
3. **Resolution**: Generate at 2x size for retina displays (e.g., if you need 400x300, generate 800x600)
4. **Transparency**: Request transparent backgrounds where possible for easier integration
5. **File Format**: PNG for illustrations with transparency, JPG for photos

## Recommended Sizes

- Hero image: 1200x900px (or larger, will be cropped)
- Service icons: 200x200px
- Team photos: 600x800px (portrait)
- Testimonial avatars: 100x100px
- Background patterns: Tileable at 400x400px minimum
- Badges: 300x300px
